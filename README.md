## Tested Environment
 
 * Node: V12
 * Mysql 8

## Unit tests
 * Please run "npm run test" to run unit test cases

 ## Steps to setup

 * Clone the project
 * If need, Update app port in 'src/main.ts' (default: 3000)
 * Create a DB. Name: 'photo_manager'
 * Update DB configurations in 'ormconfig.ts' file
 * Run "npm install"
 * Run "npm start"
